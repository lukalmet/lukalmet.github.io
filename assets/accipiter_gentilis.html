Accipiter gentilis (Linnaeus, 1758)

Famille : Accipitridae, Ordre : Accipitriformes

Statut IUCN :    Monde    France

Description :

  • Envergure : 95-105 cm (108-125 cm femelle)
  • Longueur : 48-57 cm (58-62 cm femelle)

Alimentation : autres rapaces, corvidés, divers oiseaux, petits mammifères

Habitat : grands massifs forestiers, plaine, montagne, campagnes boisées

Espèce migratrice : non

Répartition en France :

  • Résident : oui
  • Hivernant : non
  • Nicheur : non
