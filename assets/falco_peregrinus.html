Faucon pèlerin

Falco peregrinus (Tunstall, 1771)

Famille : Falconidae, Ordre : Falconiformes

Statut IUCN :    Monde    France

Description :

  • Envergure : 85-120 cm
  • Longueur : 36-48 cm

Alimentation : oiseaux capturés en vole, quelques rongeurs et insectes

Habitat : campagnes, cultures, villes, montagnes, falaises

Espèce migratrice : oui

Répartition en France :

  • Résident : oui
  • Hivernant : oui
  • Nicheur : non
