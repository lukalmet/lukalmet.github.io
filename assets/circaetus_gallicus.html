Circaète Jean-le-Blanc

Circaetus gallicus (Gmelin, 1788)

Famille : Accipitridae, Ordre : Accipitriformes

Statut IUCN :    Monde    France

Description :

  • Envergure : 170-190 cm
  • Longueur : 62-67 cm

Alimentation : serpents, lézards

Habitat : divers millieux plutôt secs avec haies, broussailles, bosquets et roches, moyenne montagne

Espèce migratrice : oui

Répartition en France :

  • Résident : non
  • Hivernant : non
  • Nicheur : oui
