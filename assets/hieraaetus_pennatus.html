Aigle botté

Hieraaetus pennatus (Gmelin, 1788)

Famille : Accipitridae, Ordre : Accipitriformes

Statut IUCN :    Monde    France

Description :

  • Envergure : 115-135 cm
  • Longueur : 50-57 cm

Alimentation : petits mammifères, oiseaux, reptiles, gros insectes

Habitat : divers milieux boisés entrecoupés de champs, large bande Sud-Ouest Nord-Est

Espèce migratrice : oui

Répartition en France :

  • Résident : non
  • Hivernant : non
  • Nicheur : oui
